# 数据分析项目作品集

面向数据分析、商业分析、数据运营与 BI 岗位的个人作品集。本仓库包含可公开展示的项目说明、核心分析代码、展示版 notebook 和 Tableau 打包工作簿；不包含个人简历或本地运行环境。

## 项目目录

| 项目 | 业务问题 | 技术栈 | 数据边界 |
| --- | --- | --- | --- |
| [GA4 Web 用户增长与转化分析](./01_GA4%20Web%20用户增长与转化分析/) | 用户在哪个转化环节流失、留存表现如何、首次获客渠道与用户结构有何差异 | BigQuery、Python/pandas、MySQL、Tableau | Google Analytics 4 公开电商演示数据 |
| [年度 Top20 损失金额及损失率产品分析](./02_年度Top20损失金额及损失率产品分析/) | 哪些产品和工序应优先核查质量损失，如何区分成本损失优先级与质量改善优先级 | Python/pandas、MySQL、Tableau | 脱敏/模拟数据，用于复现真实业务逻辑 |

## 使用说明

- 每个项目文件夹中的 `README.md` 是阅读入口，先了解业务问题、指标口径和分析边界。
- `notebooks/` 为展示版分析过程，`sql/` 为核心审计、清洗或指标复现逻辑。
- 制造业项目保留 `.docx` 项目手册，GitHub 不直接预览时可下载查看。
- 两个项目均提供 Tableau 打包工作簿，可下载后使用 Tableau 打开；原始/处理后的独立明细数据未单独公开。

## 作品集原则

两个项目都明确区分“分析发现”和“待验证建议”：不将相关性表述为因果，不将公开数据或脱敏/模拟数据的结果表述为真实业务收益。
